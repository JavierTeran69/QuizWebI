Lista de Definiciones

HTML

        Lenguaje de marcado utilizado para crear paginas web.

CSS

        Lenguaje de hojas de estilo usado para definir la apariencia de una página web.

Lista de Elementos Importantes

  1. Encabezados: Usados para definir titulos y subtitulos.
  2. Listas: Permiten organizar contenido en viñetas o numeraciones.
  3. Enlaces: Conectan diferentes páginas o recursos.
